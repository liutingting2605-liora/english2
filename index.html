📖 The Key in the Attic

推荐流程：先「🎧 盲听挑战」听两遍答大意题 → 展开原文精读（虚线词点释义）→ 开「点读」逐句跟读 → 做「读懂了吗」。

朗读使用手机自带语音引擎。若无声音：请用 Safari / Chrome 打开（微信内置浏览器可能不支持），并确认未开静音键。

🃏 记忆卡

替代 Anki 的内置复习：看正面 → 心里想答案 → 翻面 → 按真实情况选按钮。「认识」的卡间隔会翻倍（1→2→4→7→15 天），「不认识」的今天还会再来。每天最多 8 张新卡。

🔤 Unit 1 单词本

学词四步：故事里初见 → 这里集中学（点 🔊 跟读两遍，看例句）→ Anki 记住 → 第 3 站测试。周六默写前先来这里过一遍。

📚 泛读番外 · 考试题型训练

泛读要「快」：一遍读完直接做题，不逐句抠；生词能猜就猜。做完看建议用时对比一下速度。
